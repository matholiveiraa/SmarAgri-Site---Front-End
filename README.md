# SmarAgri-Site -Front-End
Site institucional acadêmico da empresa SmartAgri Technologies, contendo páginas como inicial, sobre, produtos, carrinho e login. O site apresenta de forma superficial o protótipo de um carrinho irrigador movido por inteligência artificial, com foco na apresentação da empresa.

## O que possui?
O SmartAgri Technologies é um projeto acadêmico que apresenta o site institucional de uma empresa fictícia voltada para soluções tecnológicas no setor agrícola. O site foi desenvolvido com o objetivo de simular a presença digital da empresa, contendo páginas como inicial, sobre, produtos, carrinho de compras e login de usuários, além de um rodapé institucional com informações da empresa, contatos, endereço e redes sociais.

Dentro do contexto do projeto, o site também apresenta de forma superficial e conceitual um protótipo de carrinho irrigador movido por inteligência artificial, servindo como elemento demonstrativo da proposta tecnológica da empresa. O foco principal do sistema é a apresentação institucional, a navegação entre páginas e a simulação de funcionalidades típicas de um site corporativo, não tendo como objetivo representar um sistema comercial completo ou um produto em produção.

O projeto foi desenvolvido para fins exclusivamente acadêmicos, utilizando tecnologias web como HTML, CSS, JavaScript e PHP, com execução em ambiente local por meio do XAMPP. Todas as informações e dados utilizados são fictícios, e as funcionalidades de login e cadastro possuem caráter apenas demonstrativo.
